# 普惠 AI 股息估值系统

> 多阶段股息贴现模型 · 专业金融内核 · 零门槛操作

基于 Streamlit 构建的普惠型股票估值分析系统，融合专业金融理论与AI技术，为普通投资者提供专业级的股票估值分析服务。

## ✨ 功能特点

### 📊 股票概览
- 股票代码/名称搜索
- 基本面指标展示（PE、PB、ROE、ROA、股息率、Beta等）
- 营收增长率与净利润增长率分析
- 多周期走势分析（近7日、近1个月、近6个月、近1年）
- 均线系统分析与趋势解读

### 📈 技术指标分析
- K线图与均线系统（MA5、MA20、MA60）
- RSI相对强弱指标
- KDJ随机指标
- MACD指数平滑异同移动平均线
- 布林带分析
- 技术面综合研判评分

### 🔮 走势预测分析
- AI估值结论（低估/合理/高估）
- 多模型估值对比（零增长DDM、戈登增长模型、两阶段DDM、PE/PB估值法、Graham公式、PEG估值）
- 未来60个交易日价格预测（基准/乐观/悲观情景）
- 估值模型解读与风险分析

### 💡 投资建议
- 投资价值判断与当前位置分析
- 综合分析指标展示
- 操作策略与仓位建议
- 止损止盈设置建议
- 持有周期建议

## 🎯 专业金融模型

| 模型 | 适用场景 |
|------|----------|
| **零增长DDM** | 成熟稳定型企业 |
| **戈登增长模型** | 稳定分红成长型企业 |
| **两阶段DDM** | 成长转成熟型企业 |
| **PE估值法** | 盈利能力定价 |
| **PB估值法** | 资产驱动型企业 |
| **Graham公式** | 价值投资安全边际 |
| **PEG估值** | 成长与估值匹配度 |

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name

# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app_dividend_valuation.py
```

### Streamlit Community Cloud 部署

1. 将项目推送到 GitHub
2. 访问 [share.streamlit.io](https://share.streamlit.io/)
3. 登录并点击 "New app"
4. 选择您的 GitHub 仓库、分支和主文件 `app_dividend_valuation.py`
5. 点击 "Deploy" 完成部署
